# Portfólio Pessoal - Desenvolvimento Web e Mobile

Repositório destinado à entrega da atividade prática de introdução ao HTML e CSS da disciplina **SSC0961 - Desenvolvimento Web e Mobile** do Instituto de Ciências Matemáticas e de Computação (ICMC - USP).

**Aluno:** Caio Abner Soares Araujo

## Sobre o Projeto
Este projeto consiste na adaptação e personalização de um template web estruturado para a criação de um portfólio profissional pessoal. O objetivo da atividade foi aplicar na prática os conceitos de estruturação web e as diferentes formas de aplicação de estilos (CSS inline, interno e externo).

A página já se encontra publicada através do GitHub Pages.
